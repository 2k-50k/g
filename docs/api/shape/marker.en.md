# 属性

- 详见 [图形属性](/en/docs/api/shape/api#属性)；

## attrs 绘图属性

> 通用的 [绘图属性](/en/docs/api/shape/attrs)

### x

- 中心点 x 坐标；

### y

- 中心点 y 坐标；

### r

- 几何图形的尺寸；

### symbol

- 几何图形的类型，目前支持以下 5 种常见的几何图形:
  - `circle`: 圆形
  - `square`: 正方形
  - `diamond`: 菱形
  - `triangle`: 三角形
  - `triangle-down`: 倒三角形
